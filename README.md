# AI-Driven Phishing Email Detection Using NLP

## 📌 Project Overview

This project was developed as part of my internship at the **Indian Institute of Computing and Technology (IICT)**. The project focuses on detecting phishing emails using **Natural Language Processing (NLP)** and **Machine Learning**. It includes data preprocessing, TF-IDF feature extraction, model training, performance evaluation, and a Gradio-based interface for real-time email classification.

## 🏢 Internship Details

- **Organization:** Indian Institute of Computing and Technology (IICT)
- **Project:** AI-Driven Phishing Email Detection Using NLP
- **Internship Domain:** Artificial Intelligence, Machine Learning, and Natural Language Processing

## 🚀 Key Features

- Email text preprocessing
- TF-IDF feature extraction
- Logistic Regression
- Naïve Bayes
- Random Forest
- MLP Neural Network
- Gradio web interface
- Performance comparison of machine learning models

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Gradio

## 📊 Results

- Compared four machine learning algorithms.
- MLP Neural Network achieved the highest accuracy of **96.27%**.
